# Ormi Protocol UI 
Ormi is made possible because it is built on top of DeFi giant Aave. Ormi's
lending protocol began as a fork of Aave's protocol-v2. Special thanks to the
Aave team to its commitment to technical excellence, open source, and
decentralization to make Ormi a reality.

# Ormi protocol interface

An open source interface for the decentralized liquidity protocol Ormi

Enabling users to:

- Manage and monitor their positions on the Ormi Protocol, and the overall
  status of it
- Participate on the Ormi Governance

## Configuration & Dependencies
See `package.json` for related dependencies and scripts. Run npm install to
install all the dependencies.

## Script
To run locally without docker.
```
npm run start:dev
```
Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
